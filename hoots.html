HOOTS TUTORING IS A UVA STUDENT NON-PROFIT ORGANIZATION PROVIDING AN ONLINE PEER-TO-PEER TUTORITING NETWORK.


The objective of this freelance project was to redesign the previous site to make online bookings for tutoring sessions simple and intuitive.

CHALLENGE: redesign the current website (left) to simplify the layout and facilitate easier navigation while keeping the current color scheme in mind.

ROLE: designed website UI & developed front end prototype. This freelance project took three weeks to build and is responsive to mobile interfaces. During this time, the client's logo was in the process of being redesigned by another freelancer, so the old one is featured here.
